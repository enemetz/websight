websight

websight is an iOS utility app that reads URLs, emails, phone numbers, and addresses from the camera and then lets you perform actions based on what was detected. For example, if websight sees a URL it will ask if you want to go to that URL in Safari. If it detects an email it will ask if you want to email the detected email. If the app sees a phone number it will ask if you want to call that number. If an address is found, it will ask if you want directions to the address in Maps. websight is great for being able to quickly perform actions based on what information is in front of you, just point the camera.
